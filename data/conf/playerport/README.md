# Playerport compatibility data

This folder holds generated compatibility artifacts used by the Stendhal Playerport bridge.

## Current files

- `skills.stub.json` – legacy Playerport skill names mapped to generated Stendhal IDs.
- `clans.stub.json` – clan membership import seed.
- `gods.stub.json` – gods/admin metadata seed.

These are generated from `tools/playerport-import/*` scripts.
